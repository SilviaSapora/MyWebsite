Pintos

Pintos is computer software, a simple instructional operating system framework for the 80x86 processor architecture. It supports kernel threads, loading and running user programs, and a file system, but it implements all of these in a very simple way. It was created at Stanford University by Ben Pfaff in 2004.

C, Operating System, Threads, Concurrency, Locks
